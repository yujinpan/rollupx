# vue-component-pack

js/ts/vue/scss to a library.

## Usage

### Install

```
npm install --save vue-component-pack
```

### Config

create a config file in your project.

```js
// project/vue-component-pack.config.js
module.exports = {
  outputDir: 'dist'
};
```

### Build

```shell
$ vue-component-cli
```
